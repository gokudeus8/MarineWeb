# marino-site — instruções ## Estrutura de arquivos ``` marino-site/ ├── index.html → página principal (grade de fotos + escritos recentes) ├── fotos.html → galeria completa de fotos ├── escrever.html → lista de todos os textos ├── post-exemplo.html → template de um texto (duplique para criar novos) ├── sobre.html → página sobre você ├── style.css → todo o visual do site └── fotos/ → coloque aqui todos os seus arquivos de imagem ``` --- ## Como adicionar uma foto 1. título do seu texto

trecho inicial...

 ``` --- ## Elementos disponíveis dentro de um texto ```html

frase de destaque

legenda opcional

``` --- ## Como colocar no ar (grátis) ### GitHub Pages 1. Crie uma conta em [github.com](https://github.com) 2. Crie um repositório novo (pode ser privado ou público) 3. Suba todos os arquivos 4. Vá em **Settings → Pages → Branch: main → Save** 5. O site fica em `seuusuario.github.io/nome-do-repositorio` ### Domínio próprio (opcional) Se quiser `marino.com.br` ou similar: - Compre o domínio no [Registro.br](https://registro.br) (~R$40/ano) - No GitHub Pages, adicione o domínio em Settings → Pages → Custom domain --- ## Personalização rápida Tudo que você pode trocar em `style.css` sem precisar entender CSS: ```css :root { --bg: #F5F1EB; /* cor de fundo — troque pelo hex que quiser */ --text: #1C1C1A; /* cor do texto */ --accent: #7A6A55; /* cor dos detalhes e destaques */ --border: #E0DCD4; /* cor das linhas divisórias */ } ```
